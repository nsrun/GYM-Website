# GYMFLEX - Modern Fitness Website

GYMFLEX is a professional, high-impact gym website designed with a modern dark theme and orange accents. It features a fully responsive layout and a clean, user-friendly interface.

## 🚀 Features

- **Modern UI/UX**: Dark theme with vibrant orange highlights for a high-energy fitness feel.
- **Fully Responsive**: Optimized for desktops, tablets, and mobile devices.
- **Multi-Page Layout**:
    - **Home**: High-impact hero section and key features.
    - **About**: Our mission, values, and story.
    - **Workouts**: Detailed training programs with descriptions.
    - **Contact**: Functional-looking contact form with location and social details.
- **Professional Footer**: Detailed multi-column footer with quick links, working hours, and contact information.
- **Iconography**: Integrated Font Awesome for professional visual elements.

## 🛠️ Technologies Used

- **HTML5**: For structured web content.
- **CSS3**: For custom styling, including Flexbox and Grid layouts.
- **Google Fonts**: 'Poppins' and 'Segoe UI' for modern typography.
- **Font Awesome**: For high-quality icons.

## 💻 How to Run

1. Clone this repository or download the source code.
2. Open `home.html` in any modern web browser.
3. Alternatively, run a local server (e.g., using Python: `python -m http.server 8000`) and visit `http://localhost:8000/home.html`.
